Landing Page - Barbería.
Proyecto de desarrollo frontend para una barbería profesional, enfocado en  el crecimiento digital, brindar una experiencia más profesional al usuario y un diseño responsivo.

Características:
Diseño Moderno: Interfaz limpia y atractiva utilizando HTML5 y CSS3.

Secciones: Inicio, servicios, galería y contacto.

Responsivo: Adaptado para una visualización correcta en dispositivos móviles y escritorio.
