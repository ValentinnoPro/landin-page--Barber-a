Landing Page - Barbería.
Proyecto base de desarrollo frontend para una barbería profesional, enfocado en  el crecimiento digital, brindar una experiencia más profesional al usuario y un diseño responsivo. 
Aún estamos trabajando en el proyecto para llevarle al siguiente nivel implementando lógica para que el usuario pueda seleccionar la fecha y hora del turno y tambien para que se puedan realizar pagos a traves de la página.

Características:
Diseño Moderno: Interfaz limpia y atractiva utilizando HTML5 y CSS3.

Secciones: Inicio, servicios, nosotros, precios y contacto.

Responsivo: Adaptado para una visualización correcta en dispositivos móviles y escritorio.
